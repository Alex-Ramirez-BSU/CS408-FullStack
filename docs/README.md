# Full-Stack Web Application

This repository contains a full-stack web application built with Java and Spring Boot. 
The application uses Thymeleaf for server-side templating and Bootstrap for responsive UI design. 
Data persistence and business logic are managed within the Spring Boot backend.

The project includes configuration, testing, and documentation resources to support 
development, debugging, and deployment.

## Technology Stack

- **Backend Technology Stack**
  - Language: [Java](https://www.java.com/)
  - Framework: [Spring Boot](https://spring.io/projects/spring-boot)

- **Frontend Technology Stack**
  - Templating Engine: [Thymeleaf](https://www.thymeleaf.org/)
  - UX/UI Framework: [Bootstrap](https://getbootstrap.com/)

- **Testing & Quality**
  - Unit Testing: [JUnit](https://junit.org/)
  - Integration Testing: [Spring Test](https://docs.spring.io/spring-framework/reference/testing.html)

- **Debugging & Development Tools**
  - IDE Debugger: [IntelliJ IDEA](https://www.jetbrains.com/idea/) / [Visual Studio Code](https://code.visualstudio.com/)
  - Application logs for runtime debugging
